### Project Overview
Implemented an end-to-end CI/CD pipeline to automate build and deployment
of a Maven-based Java application using Jenkins and Docker on AWS EC2.

### Tools & Technologies
- GitHub
- Jenkins
- Maven
- Docker
- AWS EC2
- Linux

### Workflow
1. Developer pushes code to GitHub
2. Jenkins automatically triggers pipeline
3. Maven builds the application
4. Docker image is created
5. Container deployed on AWS EC2

### Outcome
- Reduced manual deployment effort
- Achieved automated and repeatable deployments
